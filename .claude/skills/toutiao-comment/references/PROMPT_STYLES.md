# 头条评论风格指南

避免AI味、生成真人感评论的核心原则和示例。

---

## 核心原则

### 1. 拒绝AI套话

**AI套话特征**：
- "首先、其次、最后"等逻辑连接词
- "总的来说"、"从整体来看"等总结套话
- "值得注意的是"、"不得不说"等强调套话
- "一方面、另一方面"等平衡套话

**真人表达**：
- 直接说观点，不用逻辑连接
- 想到哪说到哪
- 不总结，说完就停

### 2. 拒绝四平八稳

**AI特征**：
- "既有利也有弊"
- "不能一概而论"
- "需要具体情况具体分析"

**真人特征**：
- 明确的立场
- 极端的观点
- 偏颇的观察

### 3. 拒绝完美表达

**AI特征**：
- 语法完美
- 用词准确
- 逻辑严密

**真人特征**：
- 口语化
- 错别字或谐音
- 不完整的句子

---

## 标点符号使用

### AI特征
- 过多使用顿号"、"
- 完美的引号配对
- 大量使用分号"；"

### 真人特征
- 用空格代替顿号
- 不用引号
- 用句号或空格分句

**示例**：
```
AI：今天天气不错、阳光明媚、是个出行的好日子。
真人：今天天气挺好 阳光明媚 出行好日子
```

---

## 用词选择

### AI常用词（避免）

**逻辑词**：
- 首先、其次、再次、最后
- 综上所述、总之、综上
- 因此、所以、因而

**强调词**：
- 值得注意的是
- 不得不说
- 确实
- 显然

**模糊词**：
- 一定程度上
- 大部分
- 许多

### 真人常用词（采用）

**语气词**：
- 嘛、呢、啊、吧
- 呗、呀、啦

**口语词**：
- 咱、俺、偶
- 咋、啥、嘛
- 挺、挺、蛮

**网络词**：
- 笑死、笑哭
- 咱就是说
- 真的会谢
- 绝绝子

---

## 句式结构

### 短句断句

**AI特征**：长句，多个分句用逗号连接

**真人特征**：短句，用句号或空格

**示例**：
```
AI：虽然这个产品有一些缺点，但是它的优点也很明显，所以我建议大家可以考虑购买。
真人：这产品有点缺点 但优点还是挺明显的 可以考虑买
```

### 不完整句子

**AI特征**：每句话都完整

**真人特征**：故意不完整

**示例**：
```
AI：我不知道该怎么评价这件事。
真人：不知道咋说这事儿
```

### 口语倒装

**AI特征**：标准语法

**真人特征**：倒装或省略

**示例**：
```
AI：我觉得这个建议不错。
真人：这建议不错我觉得
```

---

## 情绪表达

### AI特征
- 客观理性
- 情绪稳定
- 立场平衡

### 真人特征
- 明确情绪
- 立场鲜明
- 可以极端

**示例**：
```
AI：这个政策有利有弊，需要根据个人情况判断。
真人：这政策就是坑人 别信
```

---

## 语气人设

### 建议人设

**吃瓜群众**：
```
笑死 这事闹这么大
我之前就看过类似新闻
```

**老司机**：
```
这玩意儿我几年前就玩过了
确实还行 但没说得那么神
```

**吐槽役**：
```
咱就是说 这操作看不懂
明明有更好的办法不用
```

**经验党**：
```
我之前也遇到过
后来发现其实是XXX的问题
```

## 实例对比

### 例子1: 手机评论

**AI风格**：
```
首先，这款手机的性能表现出色。其次，续航能力也值得肯定。最后，考虑到价格因素，我认为这款手机具有较高的性价比。
```

**真人风格**：
```
这手机还行 续航能撑一天
性能也不卡 就是价格有点虚
```

### 例子2: 电影评论

**AI风格**：
```
这部电影在剧情设计上独具匠心，演员的表演也十分出色。虽然在节奏把控上略有不足，但总体来说是一部值得观看的佳作。
```

**真人风格**：
```
这电影看完挺感动的
演技都在线 就是前面有点拖沓
```

### 例子3: 股市评论

**AI风格**：
```
从基本面分析，这家公司的业绩增长稳定。同时，估值水平也处于合理区间。因此，建议投资者可以适当关注。
```

**真人风格**：
```
这公司业绩还行 但股价有点高了
再等等 别急着追
```

### 例子4: 生活吐槽

**AI风格**：
```
这种行为确实不太恰当。建议大家遇到类似情况时，应该通过合理的途径反映问题，而不是采取极端的方式。
```

**真人风格**：
```
这操作真看不懂
好好说话不行非得搞事
```

## 长度控制

**推荐长度**：15-50字

**示例**：

**太短**（<10字）：
```
这文章写得真好。
```

**合适**（15-50字）：
```
文章说得挺对的 我之前也遇到过类似情况
后来发现确实是这么回事
```

**太长**（>50字）：
```
这篇文章分析得非常深入，作者从多个角度进行了全面的剖析。我觉得文章的核心观点很有价值，值得大家仔细阅读和思考。同时，我也想补充一点，就是...
```

## 检查清单

发布前检查：

- [ ] 没有"首先、其次、最后"
- [ ] 没有"总的来说、综上所述"
- [ ] 没有"值得注意的是、不得不说"
- [ ] 没有长难句（单句超过20字）
- [ ] 有明确的情绪或立场
- [ ] 用词口语化
- [ ] 长度在15-50字之间

## 进阶技巧

### 1. 留白

故意不说完，让读者自己理解。

```
这事儿咋说呢 大家都懂
```

### 2. 反讽

用反讽表达观点。

```
这建议真"专业" 建议多读点书再来
```

### 3. 自嘲

用自嘲增加真实感。

```
我也就网上说说 真让我上我也怂
```

### 4. 场景代入

描述具体场景增加代入感。

```
昨天刚遇到这事儿 在地铁上折腾半天
```
