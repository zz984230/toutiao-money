# 头条活动分类

活动按多个维度分类，帮助快速识别活动类型并选择策略。

## 参与方式分类

### 1. 生成原创 (original)
**特征**：
- 要求发布原创内容（文章、微头条、视频）
- 通常有内容质量要求（字数、图片）
- 需要带特定话题标签

**收益**：现金激励、流量扶持

**示例**：发布带 #科技# 话题的微头条，赢取现金奖励

### 2. 一键参与 (one-click)
**特征**：
- 点击按钮即可参与
- 可能是抽奖或签到
- 无需内容创作

**收益**：小额现金、积分、荣誉

**示例**：每日签到活动、幸运抽奖

### 3. 点赞转发 (like-forward)
**特征**：
- 需要点赞指定内容
- 需要转发/分享到动态
- 可能需要关注作者

**收益**：小额激励、增加互动

**示例**：点赞并转发活动页，参与抽奖

### 4. 填写表单 (form)
**特征**：
- 需要填写信息（姓名、手机号、地址）
- 可能用于实物奖品发放
- 部分需要手机验证

**收益**：实物奖品、大额现金

**示例**：填写报名表单参与活动

---

## 收益类型分类

### 现金激励 (cash)
- 直接现金奖励
- 按阅读量/互动量结算
- 固定奖金

**优先级**：★★★★★

### 流量扶持 (traffic)
- 内容推荐加权
- 曝光量提升
- 粉丝增长

**优先级**：★★★☆☆

### 荣誉奖励 (honor)
- 徽章、证书
- 排行榜排名
- 创作者等级提升

**优先级**：★★☆☆☆

---

## 时间状态分类

### 进行中 (active)
- 活动时间窗口内
- 可立即参与
- 截止时间明确

### 即将开始 (upcoming)
- 已预告，未开放
- 需设置提醒

### 已结束 (ended)
- 不再接受新参与
- 可能查看结果

---

## 可行性评估

| 可行性 | 特征 | 优先级 |
|--------|------|--------|
| 高 | 网页可完成、规则明确 | ★★★★★ |
| 中 | 需要简单验证、已知模式 | ★★★☆☆ |
| 低 | 需要APP扫码、规则模糊 | ★★☆☆☆ |
| 极低 | 新类型活动、历史失败 | ★☆☆☆☆ |

## 识别关键词

页面中遇到以下关键词，可快速识别活动类型：

| 关键词 | 活动类型 |
|--------|----------|
| 发布、原创、内容 | 生成原创 |
| 参与、报名、一键 | 一键参与 |
| 点赞、转发、分享 | 点赞转发 |
| 填写、表单、信息 | 填写表单 |
| 阅读、播放、曝光 | 流量扶持 |
| 现金、奖金、激励 | 现金激励 |
| 流量、推荐、曝光 | 流量扶持 |
| 徽章、证书、排名 | 荣誉奖励 |
| 已参与、已报名 | 已完成 |
| 进行中、火热 | 可参与 |
| 已结束、已截止 | 不可参与 |
| 扫码、APP下载 | 需要移动端 |
