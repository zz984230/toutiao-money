# 提示词风格指南

避免 AI 味、创作个性鲜明内容的核心原则。

## AI 味的主要特征（必须避免）

### 结构性 AI 味

| AI 味特征 | 示例 | 正确做法 |
|----------|------|---------|
| "首先、其次、最后" | "首先，这个问题的根源在于..." | 直接切入，用自然过渡 |
| "值得注意的是" | "值得注意的是，这一现象..." | 直接说现象，不加废话 |
| "不得不说" | "不得不说，这个观点很有趣" | 直接说观点 |
| "总的来说" | "总的来说，这个趋势..." | 用观点本身结尾 |
| "一方面...另一方面..." | "一方面有利，另一方面有弊" | 直接说核心矛盾 |

### 内容性 AI 味

| AI 味特征 | 示例 | 正确做法 |
|----------|------|---------|
| 四平八稳、没立场 | "这个问题的确值得关注" | 明确站队，爱憎分明 |
| 过于完整、没情绪 | "这个现象有多个因素导致" | 选择一个角度发挥 |
| 客观、第三方口吻 | "从客观角度来看..." | 用"我"的主观视角 |
| 数据堆砌没观点 | "数据显示70%的人..." | 数据要服务于观点 |
| 喜欢教育 | "我们应该如何看待这个问题" | 直接给出看法 |

### 语言性 AI 味

| AI 味特征 | 示例 | 正确做法 |
|----------|------|---------|
| 过于工整的书面语 | "该现象反映了..." | 用口语、网络用语 |
| 长句千篇一律 | "这个问题值得深入思考" | 用具体的说法 |
| 没标点情绪 | 逗号、句号机械排列 | 用断句、感叹号传递节奏 |
| 陈词滥调 | "时间会证明一切" | 用新鲜的说法 |
| 过度礼貌 | "感谢您的提问" | 直接进入正题 |

## 反共识/争议性风格

### 核心原则

**针对主流观点提出不同看法，制造话题性和讨论点。**

### 实现方式

1. **找到主流观点的薄弱点**
   - 主流：A 很棒
   - 反共识：A 的成功是偶然的，B 才是趋势

2. **用具体案例支持观点**
   - 不要空谈"有问题"
   - 举出具体的事件/人物/数据

3. **立场要极端化一点**
   - 不要"既有利又有弊"
   - 直接说"弊大于利"或"就是骗局"

4. **用挑衅性的表达**
   - 主流："这个做法值得商榷"
   - 反共识："这个做法就是忽悠人"

### 示例

**场景**：某明星因演技上热搜

| 类型 | 内容 |
|------|------|
| ❌ AI 味 | "这位演员的进步值得肯定，相信未来会更优秀" |
| ✅ 反共识 | "这波热搜买的吧？演技能看？全是资本捧的" |

## 轻松调侃型风格

### 核心原则

**用幽默、调侃、自嘲的方式表达观点，降低攻击性但保持犀利。**

### 实现方式

1. **用夸张制造幽默**
   - 不要"这个问题比较复杂"
   - 用"这事儿能扯出十篇论文"

2. **用网络用语拉近距离**
   - "栓Q"、"我不李姐"、"大无语"

3. **用自嘲消解严肃**
   - 不要"我认为应该..."
   - 用"咱就是说"、"听我一席话"

4. **用反讽表达观点**
   - 不要"这个观点很荒谬"
   - 用"这波操作我看笑了"

### 示例

**场景**：某公司宣布裁员

| 类型 | 内容 |
|------|------|
| ❌ AI 味 | "企业调整人员结构是正常的市场行为" |
| ✅ 轻松调侃 | "上午还在画饼，下午就开始画圈圈人了，这操作我愿称之为'敏捷式收割'" |

## 提示词模板

### 反共识型模板

```
分析以下话题的热门内容，找出主流观点，然后设计一个反共识的提示词：

话题：#<话题名称>#
主流观点：[提练2-3个主流立场]

提示词要求：
1. 立场要鲜明，直接反对主流观点中的一个
2. 用具体案例或逻辑支撑观点
3. 字数50-100字
4. 必须在开头或结尾包含 #<话题名称># 标签
5. 不要AI呃（首先/其次/值得注意的是/不得不说等）
6. 用口语化表达，有情绪偏见

请生成提示词：
```

### 轻松调侃型模板

```
分析以下话题的热门内容，设计一个轻松调侃的提示词：

话题：#<话题名称>#
热门观点：[提练2-3个热门立场]

提示词要求：
1. 用幽默、调侃的方式表达观点
2. 可以用网络用语和夸张表达
3. 字数50-100字
4. 必须在开头或结尾包含 #<话题名称># 标签
5. 不要AI呃，要有人味儿
6. 可以自嘲或反讽

请生成提示词：
```

## 检查清单

在生成微头条内容前，用以下清单检查：

- [ ] 没有"首先、其次、最后"等逻辑词
- [ ] 没有"值得注意的是、不得不说"等AI套话
- [ ] 有明确的立场（爱憎分明）
- [ ] 用了口语化或网络用语
- [ ] 有情绪偏见（不四平八稳）
- [ ] 用了短句或断句制造节奏
- [ ] 包含了 #话题# 标签
- [ ] 字数在50-100字之间
